enum Google.Apis.ETagAction

Defines the behaviour/header used for sending an etag along with a request.

Enumerator
Default 

The default etag behaviour will be determined by the type of the request.

Ignore 

The ETag won't be added to the header of the request.

IfMatch 

The ETag will be added as an "If-Match" header. A request sent with an "If-Match" header will only succeed if both ETags are identical.

IfNoneMatch 

The ETag will be added as an "If-None-Match" header. A request sent with an "If-Match" header will only succeed if both ETags are not identical.

Detailed Description

template<TRequest>
class Google::Apis::Upload::ResumableUpload< TRequest >

Media upload which uses Google's resumable media upload protocol to upload data.

Media upload which uses Google's resumable media upload protocol to upload data. The version with two types contains both a request object and a response object.

See: https://developers.google.com/drive/manage-uploads#resumable for more information on the protocol.

Template Parameters:
TRequestThe type of the body of this request. Generally this should be the metadata related to the content to be uploaded. Must be serializable to/from JSON.

See: http://code.google.com/apis/gdata/docs/resumable_upload.html for information on the protocol.

Template Parameters:
TRequestThe type of the body of this request. Generally this should be the metadata related to the content to be uploaded. Must be serializable to/from JSON.
TResponseThe type of the response body.
